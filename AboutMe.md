# About This Project

This project is a self-directed IT Support & Active Directory lab designed to demonstrate real-world skills in Windows support, ticket handling, troubleshooting, automation, and documentation.

## Goals of the Lab
- Practice supporting Windows users and endpoints
- Troubleshoot Active Directory login and file access issues
- Learn how to document tickets and resolutions
- Implement basic security hardening
- Practice writing PowerShell automations
- Build a professional IT portfolio for job applications

## Technologies Used
- Windows Server 2022
- Windows 10/11
- Active Directory Domain Services
- PowerShell
- NTFS Permissions
- Security Event Logs

## What This Demonstrates
- Ability to resolve common IT support issues
- Ability to create and manage Active Directory users, groups, and shares
- Ability to automate tasks using PowerShell
- Ability to write knowledge base documentation and ticket logs
- Ability to communicate technical fixes clearly

## Why I Built This
I am pursuing IT Support as a career path and wanted hands-on practice in troubleshooting and administration in a realistic environment. This lab helps me develop practical skills needed for IT Support, Desktop Support, Endpoint Support, and Junior SysAdmin roles.
